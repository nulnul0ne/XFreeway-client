# XFreeway Client 1.0.7

Релиз XFreeway Client для Android.

XFreeway Client - мобильное приложение для подключения к инфраструктуре XFreeway: direct-нодам и cascade-маршрутам проекта. Приложение основано на Xray-core и адаптировано под подписки XFreeway Basic и Plus.

## Что внутри

- Поддержка XFreeway подписок Basic и Plus.
- Отображение доступных точек: Russia, Finland, USA.
- Direct-подключения для FI и US.
- Cascade-маршруты для Plus-подписок.
- Встроенный магазин подписок XFreeway.
- Проверка статуса сервиса Madrid/API.
- Удобный экран подключения с отображением активной точки и IP.
- Поддержка VLESS/Reality и других протоколов XrayFA.
- Material Design интерфейс, тёмная тема, импорт подписок из Mini App.
